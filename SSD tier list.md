
# stoRAGE-SSD-Buying-Guide: Tier list

`ESTIMATED "OPTIMAL PRICE RANGE" IS IN USD, ->EXTRAPOLATE<- FOR YOUR OWN CURRENCY AND FOLLOW INSTRUCTIONS. tier list instructions: BUY THE CHEAPEST DRIVE AVAILABLE FOR YOU IN THE CATEGORY THAT FITS YOUR USE CASE, DRIVES ARE IN NO PARTICULAR ORDER`

`DRIVES ARE PCIE 3.0 3D TLC NAND+DEDICATED DRAM CACHE UNLESS OTHERWISE NOTED! EOL NOTE INDICATES DISCONTUNINUITY, LIKELY NOT PRICED COMPETIVELY. IF YOU ARE CONFUSED ABOUT DRAMLESS AND QLC NOTES YOU CAN REFER TO THE WRITE-UPS LOCATED WITHIN THE MAIN GUIDE`
`If drive is marked with "PCIE 4.0" you need to be on a motherboard/cpu platform with PCIE 4.0 support to have all of the drives performance available (Drive will still function on PCIE 3.0).`
`WITH HIGHER END DRIVES AND MORE ELABORATE USE CASES, IT IS ADVISEABLE TO LOOK MORE DEEPLY AND BEYOND THIS LIST TO WEIGH WHICH
